# Course 1: Technical Support Fundamentals (Basics & Overview)

- IT Roles: Help desk → Desktop support → Sysadmin → Network admin.
- History: Abacus → Babbage/Ada Lovelace (1st programmer) → Vacuum tubes → Transistors → Integrated circuits → PCs (1970s–80s) → Moore's Law (smaller/faster/cheaper).
- Binary: Everything = 0s & 1s → 8 bits = 1 byte → 1024 bytes = 1 KB → ASCII/Unicode for letters.
- Logic Gates: AND (both 1), OR (one or both), NOT (flip), NAND/NOR common in CPUs.
- Hardware Key Parts:
    - CPU: Brain, does calculations.
    - RAM: Temporary fast storage (loses data on shutdown).
    - Motherboard: Connects everything.
    - Storage: HDD (slow/cheaper) vs SSD (fast/expensive).
    - PSU, GPU, cooling, peripherals.
- OS Basics: BIOS/UEFI → Kernel → Full OS boot.
    - Major OS: Windows (NTFS), Linux (ext4), macOS (Unix-based).
    - GUI vs CLI (terminal).
- Networking Intro: Data → packets → cables/wireless → Internet = client-server.
- Software: System (OS) vs Application; high-level languages hide binary.
- Troubleshooting: 6 steps – Identify → Theory → Test → Plan/Fix → Verify → Document.
- Customer Service: Empathy, active listening, stay calm.
