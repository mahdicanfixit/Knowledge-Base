# Course 4: System Administration & IT Infrastructure

- Sysadmin Role: Provision users/hardware, maintain servers, policies, backups, disaster recovery.
- Key Services:
    - DNS & DHCP config (manual/auto IP, records).
    - File/Print sharing, Web servers (Apache/IIS).
    - Email protocols (SMTP/POP/IMAP).
- Directory Services:
    - Active Directory (Windows domains): Users, groups, GPOs, domain join.
    - LDAP/OpenLDAP basics.
- Cloud Basics: IaaS (VMs), PaaS (platforms), SaaS (apps like Google Workspace).
- Backups: Full vs incremental → RAID (redundancy) → Test restores!
- Virtualization: Hypervisors (VMware/VirtualBox), VMs for testing.
- Best Practices: Documentation, change management, patching.
